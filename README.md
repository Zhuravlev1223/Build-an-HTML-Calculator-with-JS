# Калькулятор в горах 🏔️

Современный веб-калькулятор с красивым дизайном в стиле горного пейзажа. Реализован с использованием чистого JavaScript, HTML и CSS.

## ✨ Возможности

### Основные функции
- **Арифметические операции**: сложение (+), вычитание (-), умножение (×), деление (÷)
- **Десятичные числа**: поддержка дробных чисел с точкой
- **Очистка**: кнопка "C" для сброса всего выражения
- **Удаление**: кнопка "←" для удаления последнего символа

### Функции памяти
- **MC** - очистить память
- **MR** - вызвать число из памяти на дисплей
- **M+** - прибавить текущее число к памяти
- **M-** - вычесть текущее число из памяти

## 🎨 Дизайн

Калькулятор оформлен в стиле горного пейзажа с:
- Анимированными облаками
- Горами
- Солнцем с эффектом свечения
- Градиентными кнопками с эффектами наведения

## 🚀 Как открыть

### Способ 1: Локальный сервер (рекомендуется)
1. Убедитесь, что у вас установлен Python
2. Откройте командную строку в папке проекта
3. Запустите локальный сервер:
   ```bash
   python -m http.server 8000
   ```
4. Откройте браузер и перейдите по адресу: `http://localhost:8000`

### Способ 2: Прямое открытие файла
1. Найдите файл `index.html` в папке проекта
2. Дважды кликните по файлу или откройте его в браузере

### Онлайн-версия
- Откройте калькулятор по ссылке: https://zhuravlev1223.github.io/Build-an-HTML-Calculator-with-JS/
3. **Примечание**: некоторые функции могут не работать из-за ограничений безопасности браузера

## 📱 Как пользоваться

### Основные операции
1. **Ввод чисел**: нажимайте цифры 0-9
2. **Десятичные числа**: нажмите "." для ввода дробной части
3. **Операции**: выбирайте нужную операцию (+, -, ×, ÷)
4. **Результат**: нажмите "=" для вычисления

### Примеры использования
- **Простое сложение**: `5 + 3 =` → результат `8`
- **Дробные числа**: `3.14 × 2 =` → результат `6.28`
- **Последовательные операции**: `10 + 5 - 3 =` → результат `12`

### Работа с памятью
1. **Сохранить в память**: введите число, нажмите `M+`
2. **Вызвать из памяти**: нажмите `MR`
3. **Очистить память**: нажмите `MC`
4. **Вычесть из памяти**: введите число, нажмите `M-`

### Коррекция ввода
- **Удалить символ**: нажмите "←"
- **Очистить всё**: нажмите "C"
